Privacy Policy - Samagam Stream Scheduler

Last updated: 2026-07-17

Samagam Stream Scheduler is an internal automation tool built for and used by Gurdwara Sahib Brahm Bunga Dodra to manage YouTube live broadcasts for the organization's own YouTube channel. It is not a public-facing product and is not distributed to or used by anyone outside this organization.

What it does

The tool reads a schedule of upcoming samagams and, for the organization's own YouTube channel, automatically creates and configures scheduled live broadcasts, live streams, thumbnails, and playlists on that channel.

What data it accesses

The tool uses the YouTube Data API (https://www.googleapis.com/auth/youtube scope) to read and manage content belonging to the single, authorized YouTube channel it is connected to. It does not access, collect, or process data from any other Google account, YouTube channel, or individual user.

What data it stores

The tool stores an OAuth access/refresh token locally and, for its automated scheduled runs, as an encrypted secret in a private GitHub Actions configuration. This token is used solely to authenticate the tool's own API calls to the authorized channel. No viewer, subscriber, or third-party personal data is collected, stored, sold, or shared.

Data retention and deletion

The stored OAuth token can be revoked at any time by the channel owner via Google Account permissions, which immediately ends the tool's access.
